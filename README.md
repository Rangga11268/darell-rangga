## üéØ Project Overview

This portfolio is not just a collection of projects; it is a demonstration of **modern web engineering principles**. Designed to bridge the gap between static content and immersive interaction, this platform features a robust architecture built for performance, scalability, and user experience.

The improved interface replaces traditional static layouts with dynamic, state-driven components, offering visitors a tactile and engaging way to explore my professional background.

---

## ‚ú® Key Features

| Feature                  | Description                                                                                                                                    |
| :----------------------- | :--------------------------------------------------------------------------------------------------------------------------------------------- |
| **Interactive ID Card**  | A physics-based identity component featuring 3D tilt, holographic textures, and drag interactions, demonstrating advanced motion capabilities. |
| **AI Command Terminal**  | An integrated conversational interface powered by Google Gemini, allowing users to query professional details in natural language.             |
| **Adaptive Theming**     | A granular Light/Dark mode system that adjusts not just colors, but texturing and lighting effects for optimal readability in any environment. |
| **Command Center**       | A centralized dashboard providing quick access to system status, navigation, and live GitHub activity metrics.                                 |
| **Internationalization** | Built-in bilingual support (English/Indonesian) with seamless, state-preserved switching.                                                      |

## üõ†Ô∏è Technology Stack

Built with an industry-standard modern stack to ensure stability and maintainability.

- **Core Framework**: Next.js 14 (App Router)
- **Language**: TypeScript (Strict Mode)
- **Styling Engine**: Tailwind CSS + CSS Variables
- **motion Library**: Framer Motion
- **State Management**: React Context + Zustand
- **AI Integration**: Google Gemini API
- **Deployment**: Vercel Edge Network

## ÔøΩ Getting Started

To run this project locally for development or inspection:

1.  **Clone the Repository**

    ```bash
    git clone https://github.com/Rangga11268/my-portfolio.git
    ```

2.  **Install Dependencies**

    ```bash
    npm install
    ```

3.  **Run Development Server**

    ```bash
    npm run dev
    ```

4.  **Open Browser**
    Navigate to `http://localhost:3000`.
